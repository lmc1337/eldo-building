How to Use the Vacant Property Map

A field-ready tool for inspections, code enforcement, and data collection

Overview

What This Map Does

The Vacant Property Map is a live, editable map connected directly to the City's GIS data. Field staff can scan the QR code on any mobile device, view property conditions in real time, and update records on the spot — no return trip to the office required.

Currently in progress: All points for unused meters have been loaded. The north side of the city still requires field verification to confirm vacancy status.

Icon Legend

Color Indicates Condition

Each property is color-coded based on the severity of any code violations on record.

Green — No Violations

Property is in acceptable condition with no outstanding code issues.

Orange — Code Violation

An active code violation has been identified and requires attention.

Red — Potentially Condemnable

Serious structural or safety concerns that may warrant condemnation proceedings.

Occupancy Status

Shape Indicates Vacancy

The shape of each icon tells you whether the property is currently occupied.

Empty Triangle

Unoccupied

Solid Dot

Occupied

Question Mark

Unknown

Editing Data

Making Updates in the Field

The left-side panel provides access to the property editor. Select any point on the map to view its details, then use the editor to update vacancy status, condition, or code violation severity. A dropdown at the top of the editor panel also provides access to the full map legend.

Use caution when editing. This map writes directly to the City's live GIS data. Deleted records are difficult to recover. Take care when working on smaller phone screens — accidental taps can modify or remove data.

Device Compatibility

Best Practices

This application is optimized for mobile phones and desktop browsers. A tablet provides the best field experience, offering more screen space for the editor and map simultaneously. Display may vary across phone screen sizes — if the interface feels cramped, try rotating to landscape orientation.
